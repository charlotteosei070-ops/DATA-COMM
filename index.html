Topic Title

Understanding the Telegraph

Fig 1: Telegraph Key - The First Digital Communication Device

How a Layman Understands the Telegraph

1

The Button (Key)

Think of it like a doorbell button. When you press it, electricity flows. When you release it, electricity stops. Simple on/off!

2

The Wire

Just like a garden hose carries water, the wire carries electricity from point A to point B across long distances.

3

The Sounder

At the other end, a metal bar clicks when electricity arrives. Short click = dot (·), Long click = dash (–).

4

Morse Code

Patterns of dots and dashes represent letters. SOS = ··· ––– ···. It's like texting, but with beeps!

Introduction to Data Communication

Data communication is the exchange of data between two devices via some form of transmission medium such as wire cable, fiber optics, or wireless signals.

Sender

Device that sends the data message

Message

Information or data to be communicated

Medium

Physical path for transmission

Receiver

Device that receives the message

Protocol

Set of rules for communication
